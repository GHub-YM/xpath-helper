# XPath Helper Extension

一个简洁好用的浏览器扩展，帮助 **快速定位网页元素的 XPath 路径**。

## 📦 安装方法

### 方法一：从源码安装（适合开发者）
1. 下载并解压。
2. 启动浏览器。
3. 打开浏览器的"管理扩展"：
4. 打开右上角 **开发者模式 (Developer mode)**。
5. 点击 **加载已解压的扩展程序 (Load unpacked)**，选择本项目文件夹即可安装。

## 🛠 使用说明
安装完成后，点击浏览器右上角的插件图标，即可启动 **XPath 辅助窗口**。
